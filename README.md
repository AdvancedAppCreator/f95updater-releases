# f95updater-releases

Public releases and help documentation for **F95 Updater**, the Android
companion app for tracking F95Zone game updates across installed Android apps
and JoiPlay libraries.

- Download the latest APK from
  [GitHub Releases](https://github.com/AdvancedAppCreator/f95updater-releases/releases/latest).
- Read the help site at
  [advancedappcreator.github.io/f95updater-releases](https://advancedappcreator.github.io/f95updater-releases/).
- Report issues at
  [GitHub Issues](https://github.com/AdvancedAppCreator/f95updater-releases/issues).
