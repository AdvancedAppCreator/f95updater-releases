# Change a wrong match

Auto-match is conservative but not perfect. If a row was mapped to the wrong
F95 thread, fix it in one step.

## Steps

1. Tap **✏️** on the row.
2. In the **Search catalog** box, type the correct title.
3. Tap the right result.

That's it — the mapping is overwritten with the new thread, version, and
canonical URL, and a snackbar confirms `Mapped to '<correct title>'`.

## Alternative: paste the correct URL

If you already have the right thread URL open in your browser, just paste it
into the **F95Zone thread URL** field and tap **Save URL**. See
[Paste a thread URL](paste-url.md).

## What if I want to remove the mapping entirely?

Tap **Clear URL**. The row goes back to ⚪ "Not mapped" status, and the next
**Refresh from catalog** will try to re-match it.

If you want it to **stop** trying to re-match, use [Not on F95](not-on-f95.md)
instead.
