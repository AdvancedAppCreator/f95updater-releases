# Install a game in JoiPlay

Convenience action that hands an archive (zip / rar / 7z / tar) to JoiPlay's
own importer, without you having to switch apps and re-pick the file.

## How to use

1. Download the game archive (typically from the F95Zone thread, **Android**
   port section).
2. In F95 Updater: menu → **JoiPlay** → **Install game in JoiPlay…**.
3. Pick the archive.

JoiPlay opens, prefilled with the archive — finish the import there as
usual.

## Notes

- This **does not** modify your archive or install the game itself. We're
  just passing the URI to JoiPlay via an intent.
- JoiPlay must be installed for this option to be useful. If JoiPlay isn't
  present, you'll get a snackbar error.
