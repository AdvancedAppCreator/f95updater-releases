# Review unmapped games

Review unmapped games helps you resolve rows that the automatic matcher skipped
or could not safely match.

## Where to find it

Menu -> **Catalog** -> **Review unmapped games**.

## What it shows

The dialog lists currently unmapped rows and any safe suggestions the app found.
For each row you can:

- Pick a suggested F95 catalog match.
- Search manually.
- Mark the row as **Not on F95**.
- Skip it for now.

## Manual matches

Manual choices are saved as manual mappings. They are protected during normal
catalog refreshes unless you enable **Overwrite manual matches on refresh** from
the Catalog submenu.

## Tips

Use this after importing a JoiPlay backup or after a catalog sync adds many new
games.
