# F95 Updater Help

A small Android companion app that tracks **F95Zone** game updates for apps
installed on your phone and for games imported from a **JoiPlay** backup.

## What this app is — and isn't

- ✅ Lists Android apps and JoiPlay games, **shows the latest F95 version**,
  and flags ones that have an update available.
- ✅ Lets you open the F95Zone thread for each game with one tap.
- ❌ **Does not** automatically download or install game APKs. You always go
  through the F95Zone thread to fetch new builds yourself.

## Reporting a problem

Open the menu → **Diagnostics** → **Save logs to Documents** (or **Upload app
logs** if you have logging configured), then describe what you did. See the
[Diagnostics page](diagnostics/logs.md) for details.
