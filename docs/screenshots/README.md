# Screenshots

Drop PNGs in this directory. They're referenced by `![alt](screenshots/name.png)`
from the docs pages.

## Filename conventions

`<feature>-<state>.png` — e.g. `mapping-edit-dialog.png`,
`joiplay-delete-confirm.png`.
